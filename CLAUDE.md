# Agentic AI Playbook — Learning Project

This is a self-study repo. I (the user) am learning agentic AI from the 5 books in `books/`, following the order and plan in `LEARNING_PATH.md`. Day-wise reading plans live inside each numbered practice folder (e.g. `01-langchain-basics/READING_PLAN.md`).

## The Learning Process — ALWAYS follow this

When I come to this project to learn, we follow this exact flow for every page/section of a book:

1. **I read** the page/section in the PDF myself first.
2. **I tell you** in 2–3 lines what I understood (before seeing your version). Encourage me to do this if I skip it — it's retrieval practice.
3. **I paste** the page text into the chat.
4. **You give two layers:**
   - ⚡ **Quick summary** — 3–4 bullets, the skeleton of the page/section
   - 🔍 **Detailed view** — explanation of each important point, the "why", and anything tricky I might have missed
5. **We compare** my understanding with yours — point out my gaps honestly.
6. **After we discuss and I confirm**, you update the day-notes md file (e.g. `01-langchain-basics/day-01-chapter-1-notes.md`) with the agreed section note. I write my own personal notes separately in my physical notebook.

## Rules for Claude

- **Never write notes to files without discussing first and getting my confirmation.**
- Notes should be **per section, not per page** — short skeleton notes (3–4 bullets) plus key numbers/terms worth quoting exactly. No long copies of the book.
- Explain in **simple English** (I'm a non-native speaker), but always keep concrete details: stats, key terms, core design ideas.
- Push me to answer end-of-chapter Questions and to rebuild code examples from memory — don't let me read passively.
- When a chapter is done, remind me to: write chapter bullets in `notes/`, tick the checkbox in the book's `READING_PLAN.md`, and update the tracker in `LEARNING_PATH.md`.
